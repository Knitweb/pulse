Feed explorer — don't trust, verify

recomputes the Merkle root over every record in your tab (canonical CBOR + SHA-256, the exact bytes knitweb.fabric.feed signs) and compares it against the published head. monitor
The root check proves the records ARE the set the head commits to. The head's secp256k1 signature itself is one line with pulse:
python3 -c "import json,urllib.request as u; from knitweb.fabric.feed import FeedHead, verify_head; h=json.load(u.urlopen('HEAD_URL')); print(verify_head(FeedHead(**h)))"
